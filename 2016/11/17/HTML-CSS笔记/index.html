发表于 2016-11-17

##标签
<title> </title> 标题标签 显示在浏览器选项卡上
<p></p> 段落标签
<h1></h1> <h6></h6> 1级标题-6级标题 字号逐渐减小
<strong></strong> 强调 显示为粗体
<em></em> 强调 显示为斜体
<q></q> 短文本引用 会给引用内容加上双引号
<blokquote></blockquote> 长本文引用 浏览器解析时候会缩进
<br/> 换行 空标签 没有HTML内容的标签
&nbsp; 空格
<hr/> 水平横线 horizontal
<address></address> 地址 解析为斜体
<code></code> 单行代码
<pre></pre> 多行代码

<div></div> 独立逻辑 板块
<div id="版块名称">…</div> 给div命名
<a href="www.baidu.com" title="点击进入首页">click here!</a> 超链接
<a>标签默认在当前窗口打开链接
使用<a href="网址" target="_blank">click here在新窗口打开
<img src="" alt="" title=""> src指向地址 alt占位图片 title鼠标滑过显示的文本
